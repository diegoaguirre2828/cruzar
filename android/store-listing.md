## Post-launch — set this env var on Vercel

Once the listing is live, grab the Play Store URL (looks like
`https://play.google.com/store/apps/details?id=app.cruzar.twa`) and set:

```
NEXT_PUBLIC_TWA_PLAY_STORE_URL=https://play.google.com/store/apps/details?id=app.cruzar.twa
```

This flips on the `TwaPromoBanner` on cruzar.app for Android web users,
nudging them toward the Play Store install instead of the Chrome A2HS
flow.
